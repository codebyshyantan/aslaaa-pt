# Export System

Exports are generated in the frontend with `xlsx` / SheetJS.

## Workbook sheets

- `Lobby PT`
- `Merge PT`
- `Weekly PT`
- `Tournament PT`
- `Unique Teams`
- `Import Reports`
- `Automation Runs`
- `Suggestions`
- `Achievements`
- `Snapshots`

## Data sources

- live lobby entries
- saved merge presets
- immutable daily snapshots
- unique team registry
- activity log import events
- automation run history

## Output file

- `aslaaa-pt-exports.xlsx`
